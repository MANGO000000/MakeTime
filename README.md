# ⏱️ Pomodoro Timer – Java Swing Productivity App

전주대학교 2학년 2학기 소프트웨어 프로젝트 VI 자바 Java Swing 기반의   **뽀모도로 타이머** 프로젝트입니다.  
공부 카테고리를 선택하고, 25/5·50/10·90/30 세션 또는 사용자 지정 타이머를 실행할 수 있으며,  
기록은 CSV 형태로 자동 저장되고, JFreeChart를 통해 **일간·주간·월간 그래프**로 시각화할 수 있습니다.

---

## 🖥️ 주요 기능 (Features)

### ✔️ 1. 뽀모도로 타이머 기능
- 25분 집중 / 5분 휴식  
- 50분 집중 / 10분 휴식  
- 90분 집중 / 30분 휴식  
- 사용자 지정 세션 (예: 40/10)
- Start / Pause / Reset 지원
- 공부 세션 종료 후 자동 휴식 전환

---

### ✔️ 2. 공부 카테고리 관리
- 카테고리 추가 (예: 토익, 코딩, 자격증 등)
- 콤보박스에서 선택하여 기록 저장 시 자동 반영

---

### ✔️ 3. 학습 기록 자동 저장 (CSV)
- 저장 경로: `/records/record.csv`
